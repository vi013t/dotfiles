# dotfiles

Violet Iapalucci's dotfiles, including configuration for:

- Darling
- Joshuto
- Neovim
- OneDrive
- VSCodium
- Wezterm

## Installation

To use these dotfiles, clone it and move its contents into `~/.config`:

```bash
git clone https://github.com/vi013t/dotfiles.git
cd dotfiles
mv * ~/.config
```
