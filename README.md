# dotfiles

Violet's dotfiles, including configuration for:

- Darling
- Joshuto
- Neovim
- OneDrive
- VSCodium
- Wezterm

## Installation

To use these dotfiles, clone it and move its contents into `~`:

```bash
git clone https://github.com/vi013t/dotfiles.git
cd dotfiles
mv * ~/
darling all load-installed
```
